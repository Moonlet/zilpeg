# Zilliqa Peg proof of concept

This proof of concept is using [LotionJS](https://lotionjs.com) framework.

## Project setup

### Requirements:
- node v7.6.0 or higher
- macos or linux (didn't test on windows)

### Insall dependencies:
```bash
npm install
```

## Start the project
- Start the peg zone blockchain
```bash
npm start
```

- Get current state
```bash
npm run state
```

- Do a transaction
```
npm test
```
